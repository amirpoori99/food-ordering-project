- **Security:** JWT Authentication, BCrypt encryption
- **API:** RESTful services with JSON
- **Testing:** JUnit 5, Mockito, AssertJ
- **Build:** Maven 3.x

- **Architecture:** MVC pattern

## 🛠️ **Installation & Setup**

### **Prerequisites**
- Java 17 or higher
- Maven 3.6+
- Git

### **Backend Setup**
```bash
cd backend
mvn clean compile
mvn test                    # Run all backend tests (1,850+ tests)
mvn spring-boot:run        # Start backend server
```

### **Frontend Setup**
```bash
cd frontend-javafx
mvn clean compile
mvn test                    # Run all frontend tests (650+ tests)
mvn javafx:run             # Start JavaFX application
```

### **Full System Test**
